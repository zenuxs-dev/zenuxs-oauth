What this demo uses

The only required config is clientId. Everything else is optional.

Config

{
  "clientId": "f13975eff3604b2e",
  "storage": "localStorage",
  "theme": "dark",
  "uiFallbackMode": "popup",
  "debug": true
}

Notes

  • Open this example from a local web server, not file://.
  • oauth.init() runs on load and auto-finishes callbacks.
  • Google, Discord, and GitHub can be moved into popup mode if the embedded sheet cannot continue.
  • The dedicated callback route example lives in callback.html.
